## НАЗНАЧЕНИЕ ПРОГРАММЫ

Эта программа предназначена для фильтрации различных типов данных (целых чисел, вещественных чисел и строк) из множества файлов и записи их в отдельные выходные файлы. Программа работает из командной строки и позволяет гибко настраивать параметры фильтрации и сохранения результатов.

## ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ
Учитывая что программа учебная и предоставляются исходные данные, Вам необходимо собрать артефакт jar.
Для работы программы вам необходимо разместить файлы на локальном хранилище и предоставить программе доступ.
Далее описан алгоритм работы с программой.
1.Ввод файлов:
2.При запуске программы она предложит вам ввести имена файлов для фильтрации. Ввод осуществляется построчно. Когда вы закончите ввод, напишите команду "обработать файлы" для начала обработки.Пример:
3.Введите имена файлов (по одному в строке). Завершите ввод командой 'start':
4.file1.txt
5.file2.txt
start


Программа поддерживает опции, следующие опции которые необходимо указывать в командной строке вместе с запуском программы из терминала.
-o, --output-dir: Указать путь к директории для выходных файлов. По умолчанию текущая директория .
-o /path/to/output/
-p, --prefix: Задать префикс для имен выходных файлов. По умолчанию пустой.
-p my_prefix_
-a, --append: Включить режим добавления в существующие файлы. По умолчанию файлы перезаписываются.
-s, --short-statistics: Показывать краткую статистику (количество элементов в каждом типе данных). По умолчанию полная статистика


Выходные файлы создаются с именами:
integers.txt для целых чисел
floats.txt для вещественных чисел
strings.txt для строк


Перед первым запуском убедитесь, что у вас установлены все необходимые зависимости. Программа использует библиотеки Apache Commons CLI, Apache Commons IO и Picocli. Все они автоматически загружаются через систему сборки Maven.
